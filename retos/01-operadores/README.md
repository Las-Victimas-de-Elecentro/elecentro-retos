# #01 OPERADORES

## Ejercicio

```
¡Mis queridas y estimadas cabras! Hoy tenemos otro reto bastante sencillo, pero no
se confíen: las cosas se irán complicando a medida que avancemos.

Antes de empezar, ¿tienen claro lo que es un operador?

Estos son símbolos o palabras reservadas que nos permiten realizar una operación,
ya sea matemática, de comparación, de evaluación lógica, de identidad... Incluso hay
otros operadores más "especiales", como de pertenencia en un conjunto o manipulación
de bits.

De seguro te llevas una sorpresa al ver operadores que no conocías.

EJERCICIO:
- Crea ejemplos utilizando TODOS los tipos de operadores de tu lenguaje. Ya sean
  aritméticos, lógicos, de asignación, comparación... TODOS los que se pueda.
- Debes mostrar por consola todos los ejemplos.

DIFICULTAD EXTRA (opcional):
Crea un programa que solicite dos números y muestre TODAS LAS OPERACIONES POSIBLES que
se pueden hacer con ellos en tu lenguaje, utilizando únicamente sus operadores.

(Hey, hey. Validaciones para el que pueda hacerlas. Apenas estamos calentando, chaval)
```
